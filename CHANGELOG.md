# Changelog

All notable changes to BODHI will be documented in this file.

## [0.1.4] - 2025-01-19

### Documentation
- Updated README with v0.1.3 prompting features
- Added all 19 authors to PyPI metadata
- Expanded Contributing section with future directions
- Added GitHub Actions workflow for automated releases

## [0.1.3] - 2025-01-18

### Added
- **Task Type Routing**: CONVERSATION, TECHNICAL, HYBRID, EMERGENCY
- **Audience Detection**: PATIENT, HEALTH_PROFESSIONAL, UNCLEAR
- **BE SPECIFIC**: Include dosages, frequencies, timeframes
- **ACTIVELY ASK**: Direct questions instead of passive conditionals
- **INCLUDE ALTERNATIVES**: Mention multiple options
